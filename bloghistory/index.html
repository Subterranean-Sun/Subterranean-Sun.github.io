# 2022

# 前期

  • 在访问 B 站、CC98、知乎等诸多网站以及搜索资料时均发现很多的网友都拥有属于自己的一套 Blog，很多采用了十分美观简洁大气的主题，观赏体验极佳，遂萌生了建造自己 Blog 的想法，其中东方鬼畜音 Mader@拔剑 D-Sketon 的 Blog 对我的影响最大，他使我了解到了 Hexo 框架和 Shoka 主题。

# 8 月

# 4 日
  • 这一天之后到后来的几天一直都很闲，遂在 B 站搜索视频查看教程，同时配合上网上搜到的其他教程去搭建。
  • 首先下载 Nodejs 和 Git，虽然现在不知道有什么用然后一顿操作猛如虎，搞了 Hexo 和 Shoka 主题，最后发现用本地输入方法能展示 Blog，但没有办法修改，查找一下错误才发现是 Git Bash 内的用户名输错了，此时已经是次日凌晨两三点了，已经不想现在重做了，Blog 也就暂且搁置。
# 5 日
  • 这一天我仍然犯了不少错误，比如仓库名输错最终费了半天，终于从一个初学者角度建立了一个 Blog，并且还可以本地修改，通过 Git Bash 上传到 GitHub 仓库进行更新。
  • 搭载了 Shoka 主题，Blog 一下子简洁大方了起来，这里特别感谢 Shoka 主题的作者 @優萌初華，作为一名医学生，能自己搭建一个漂亮的主题，实属不易。不过还是稍微吐槽一下主题作者本人的博客刷新过于阴间，很影响观感。
  • 基于 GitHub 仓库搭建的配上 Shoka 主题的 Hexo 博客已经有了雏形，在主题提供者的引导下，完成了图片替换、音乐替换、文字替换、友链添加等多项工作，效果不错，同时为了提高一点逼格，买了域名搭载到 Blog 上。
# 6 日
  • 主要集中在完善架构上。注意到了 Blog 的 https 未生效，通过重新解析域名 DNS 是叫这个吧，反正我很业余解决了问题。
  • 文章的评论区和阅读次数有冲突性 bug，关闭评论会导致无法查看阅读次数，这一天未能解决。
  • 上传了一篇很烂的数学笔记，发现公式渲染仍然有问题，不知是书写的问题还是其他问题之后只能截图保证正确性吗？...
    这一天改动的 bug 不多，但让我进一步摸清了 Shoka 主题配置，同时感谢友链内人员提供的诸多解决方案，对我 Blog 的完善帮助很大。
# 7 日
  • 对 footer.njk 文件中的 HTML 代码进行修改来确保阅读次数显示正常，引入了简洁的 busunazi（不蒜子）计数器解决了问题。毕竟计数和评论区是 Valine 的魔改的魔改，很多都不一样了嘛。
  • 在查看同主题 Blog 并学习借鉴时，通过查阅网页源码学习了如何搞定网页计时器以及网页查看计数，效果还能看得过眼。
  • 对整个 Blog 的内容继续修改，同时首次更新该文章记录 Blog 历程。以后可能很难日更，不过会持续更新的。
  • 修改了部分随机图片显示不全导致的观感问题。但没有完全修改过来
# 8 日
  • 主要针对随机图库进行大改，参考了 @灵香轩的修改方式，使得随机图库观感更好。另外对图库进行更新，尽量避免显示不全。
  • 今天被 git 坑了，Blog 差点要重开了... 不过最后还是大致恢复了。
# 9 日
  • 今天换了个上传方式搞随机图库，以前的不是加载慢不出来就是上传不成功，另外想给 Blog 网站提速却发现好像必须要云服务器，又要掏钱又要备案一大堆的，来个懂哥指点一下？
  • 想把 Blog 部署到 Coding 上却发现 Coding 好像不像 GitHub 一样支持仓库的
  • 学校的 edu 邮箱太坑了，给 git 换了个 Outlook 邮箱。
# 13 日
  • 前几天就基本没管过 Blog 了毕竟我现在没啥要搞的，不过还是细化了某些地方，比如底边计数还有图片效果等等。
# 18 日
  • 厚礼蟹，暑假应该留校找个项目搞的。。。
  • 好像有办法把公式渲染的错误修改，还是要规范写 markdown 的。
