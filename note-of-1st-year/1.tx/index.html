例1

$z=xy$图像为$z=x^{2}-y^{2} \,$旋转而来，原来的$y={ }_{-}^{+} x$移动到$x,y$坐标轴所得，则 $z=x y$与$x+y+z=1$所包围区域投影为$x+y=1$在第一象限区域，且中间依靠$y=\tfrac{1-x} {1+x} \,$区分，则所围区域体积为：
其中$D1$: $0 \leqslant x \leqslant 1,0 \leqslant y \leqslant \frac{1-x} {1+x},\\ D 2: 0 \leqslant x \leqslant 1, \frac{1-x} {1+x} \leqslant y \leqslant 1-x\$
$V_{1}=\int_{0}^{1} x d x \int_{0}^{\frac{1-x} {1+x} } y d y=\frac{1} {2} \int_{0}^{1} x\left(\frac{1-x} {1+x} \right)^{2} d x \$
令$\frac{1-x} {1+x}=t \Rightarrow x=\frac{1-t} {1+t} \,$,Then $V_{1}=\int_{0}^{1} \frac{(1-t)t^{2} } {(1+t)^{3} }dt \,$ ，
令$1+t=u$,$V_1=\int_{1}^{2} \frac{(u-1)^{2}(2-u)} {u^{3} } d u=\int_{1}^{2} \frac{-u^{3}+4 u^{2}-5 u+2} {u^{3} } d u=-u+4 \ln u+\frac{5} {u}-\frac{1} {u^2}|_1^2=4\ln(2)-$$\frac{11} {4} \,$
$V_{2}=\int_{\frac{1-x} {1+x} }^{1+x}(1-x-y) d y=y-xy-\tfrac{1} {2}y^2|_{\tfrac{1+x} {1-x} }^{1-x}=(1-x)^2\tfrac{x^2} {2(1+x)^2} \,$令 $1+x=t$ ,
, $V_{2}=\frac{1} {2} \int_{1}^{2} t^{2}-6 t+13-\frac{12} {t}+\frac{4} {t^2} d t=\frac{25} {6}-6 \ln 2 \Rightarrow I=\frac{17} {12}-2 \ln 2$

例2

$\iint_{D} \sqrt{\frac{2 x-x^{2}-y^{2} } {x^{2}+y^{2}-2 x-2} } d x d y, D: x^{2}+y^{2}=2 x, y=x-1, y=0 \text { also } \left((x-1)^{2}+y^{2}=1\right)$
令
$x-1=r\cos \theta,y=r\sin\theta,\theta \in [0,\frac{\pi} {4}],r\in[0,1]$
$I=\int_{0}^{\frac{\pi} {4} } d \theta \int_{0}^{1} \sqrt\frac{1-r^{2} } {1+r^{2} } r d r$
Let $r^2=t$,
$I=\int_{0}^{1} \frac{\pi} {4} \sqrt\frac{1-t} {1+t} \frac{1} {2} d t=\frac{\pi} {8} \int_0^1\frac{1-t} {\sqrt{1-t^2} }dt=\frac{\pi} {8}(\arcsin t+{\sqrt{1-t^2} })=\frac{\pi} {8} \left(\frac{\pi} {2}-1\right)$

0\leqslant r \leqslant 2 \cos (\phi)\\
0 \leqslant \phi \leqslant \frac{\pi} {4} \\
0 \leqslant \theta \leqslant 2 \pi
\end{matrix} \right.$
和$
\left\{\begin{matrix}
0 \leqslant r \leqslant \frac{\cos (\phi)} {\sin ^{2}(\phi)} \\
\frac{\pi} {4} \leqslant \phi \leqslant \frac{\pi} {2} \\
0 \leqslant \theta \leqslant 2 \pi
\end{matrix} \right.$

例4

计算$\iiint_{\Omega} \frac{1} {(1+x+y+z)^3} \,$,其中$x=0,y=0,z=0,x+y+z=1$围成,$\Omega$:$
\left\{\begin{array} {l}
0\leqslant z\leqslant1-x-y\\
(x,y)\in D
\end{array} \right.
\left\{\begin{array} {l}
0\leqslant x\leqslant1\\
0\leqslant y\leqslant1-x
\end{array} \right.
$

原来的式子
$=\iint_Ddxdy\int_0^{1-x-y} \frac{1} {(1+x+y+z)^3}dz=\iint_D-\frac{1} {2} \frac{1} {(1+x+y+z)^2}|_0^{1-x-y}dxdy=-\frac{1} {2} \int_0^1dx\int_0^1-x(\frac{1} {4}-\frac{1} {(1+x+y)^2})dy=\frac{1} {2} \ln (2)-\frac{5} {16}
$

例6

计算$I=\iiint_Vz^2dxdydz$,其中V为$x^2+y^2+z^2=R^2$与$x^2+y^2+z^2=2Rz$围成

解法1

由$
\left\{\begin{array} {l}
x^2+y^2+z^2=R^2\\
x^2+y^2+z^2=2Rz
\end{array} \right.
$
$\Rightarrow$
$\left\{\begin{array} {}
z=\frac{R} {2} \\
x^2+y^2=(\frac{\sqrt 3} {2}R)^2
\end{array} \right.
\\V=V_上+V_下=\iiint_{V_{上} }z^2dV+\iiint_{V_{下} }z^2dV\\
=\int_\frac{R} {2}^R\pi(R^2-z^2)z^2dz+\int_0^\frac{R} {2} \pi(2Rz-z^2)z^2dz=\frac{59} {480} \pi R^5$

方法2

$\tan\phi=\frac{\tfrac{\sqrt{3} } {2}R} {\tfrac{R} {2} }=\sqrt{3},\phi=\frac{\pi} {3} \,$
$V=V_{锥上}+V_剩,V_{锥上}:0\leqslant \rho\leqslant R,0\leqslant\phi\leqslant \frac{\pi} {3},0\leqslant\theta\leqslant2\pi\\
V_剩:0\leqslant \rho\leqslant 2R\cos\phi,\frac{\pi} {3} \leqslant\phi\leqslant\frac{\pi} {2} ,0\leqslant\theta\leqslant2\pi$
原式=$\int_0^{2\pi}d\theta\int_0^{\tfrac{\pi} {3} }d\phi\int_0^R\rho^2\cos^2\phi\rho^2\sin\phi d\rho$$+\\
\int_0^{2\pi}d\theta\int_{\tfrac{\pi} {3} }^{\tfrac{\pi} {2} }d\phi \int_0^{2R\cos\phi} \rho^2\cos^2\phi\rho^2\sin\phi d\rho=\frac{59} {480} \pi R^5\\
=\int_0^{2\pi}d\theta\int_0^{\frac{\pi} {3} } \cos^2\phi \sin\phi d\phi\int_0^R\rho^4 d\rho+\int_0^{2\pi}d\theta\int_{\tfrac{\pi} {3} }^{\tfrac{\pi} {2} } \cos^2\phi \sin\phi d\phi\int_0^{2Rcos\phi} \rho^4d\rho$

例7

求$I=\int_{-1}^1dx\int_0^{\sqrt{1-x^2} }dy\int_1^{1+\sqrt{1-x^2-y^2} } \frac{dz} {\sqrt{1-x^2-y^2} } \,$
$V=
\left\{\begin{array} {}
1\leqslant z\leqslant 1+\sqrt{1-x^2-y^2} \\
0\leqslant y\leqslant \sqrt{1-x^2} \\
-1\leqslant x\leqslant 1
\end{array} \right.
$
$y=\sqrt{1-x^2} \Rightarrow x^2+y^2=1(y\geqslant 0)$
$z=1+\sqrt{1-x^2-y^2} \Rightarrow x^2+y^2+(z-1)^2=1$上半球面
解:$V:\frac{1} {\cos\phi} \leqslant\rho\leqslant 2\cos\phi
,0\leqslant \phi \leqslant\frac{\pi} {4},0\leqslant \theta \leqslant \pi$原式
$=\int_0^\pi d\theta \int_0^{\tfrac{\pi} {4} }d\phi\int_{\tfrac{1} {\cos\phi} }^{2\cos\phi} \frac{1} {\rho} \rho^2\sin\phi d\rho=\frac{1} {2} \int_0^{\pi}d\theta \int_0^{\frac{\pi} {4} } \sin \phi(4\cos^2\phi-\frac{1} {\cos^2\phi}) d\phi
=\pi[-\frac{2} {3} \cos^3\phi - \frac{1} {2\cos\phi}]|_0^{\frac{\pi} {4} }=(\frac{7} {6}-\frac{2\sqrt{2} } {3})\pi$

例8

计算$I=\int_0^1dx\int_x^1dy\int_y^1ye^{z^4}dz$
$V:y\leqslant z\leqslant 1,x\leqslant y\leqslant 1,0\leqslant x\leqslant 1$，下曲面$z=y$,上曲面$z=1$

解法1

$I=\int_0^1dx\int_x^1dy\int_y^1e{z^4}dz=\iiint_V ye^{z^4}dV=
\int_0^1dz\int_0^zdy\int_0^yye^{z^4}dx=\int_0^1e^{z^4}dz\int_0^zy^2dy=\frac{1} {3} \int_0^1z^3e^{z^4}dz=\frac{1} {12} \int_0^1e^{z^4}d(z^4)$

解法2

$I=\int_0^1dx\int_x^1dy\int_y^1ye^{z^4}dz=\int_0^1dy\int_0^ydx\int_y^1ye^{z^4}=\int_0^1ydy\int_y^1ye^{z^4}dz=
\int_0^1dz\int_0^zy^2e^{z^4}dy=\frac{1} {3} \int_0^1z^3e^{z^4}dz=\frac{1} {12} \int_0^1e^{z^4}d(z^4)=\frac{1} {12}(e^{z^4})|_0^1=\frac{1} {12}(e-1)$

例9

求$\int_c(x^{\frac{4} {3} }+y^{\frac{4} {3} })ds$,其中c为内摆线$x^{\frac{2} {3} }+y^{\frac{2} {3} }=a^{\frac{2} {3} }(a>0)$的弧
解：曲线第一象限曲线$c_1$:$
\left\{\begin{array} {}
x=a\cos^3t\\
y=a\sin^3t
\end{array} \right.(0\leqslant t\leqslant \frac{\pi} {2})
$曲线关于x轴，y轴对称。被积函数关于y\x为偶函数，只需计算第一象限曲线积分乘以4即可。故原式$=4\int_{c_1}(x^{\frac{4} {3} }+y^{\frac{4} {3} })ds=4a^{\frac{4} {3} } \int_0^{\frac{\pi} {2} }(\cos^4t+\sin^4t)3a\cos t\sin t=6a^{\frac{7} {3} } \,$

例10

计算$\int_L(x^2+2y)ds$，其中L为曲线$
\left\{\begin{array} {}
x^2+y^2+z^2=R^2\\
x+y+z=0
\end{array} \right.
$
解：由轮换对称知，$\int_Lx^2ds=\int_Ly^2ds=\int_Lz^2ds=\frac{1} {3} \int_L(x^2+y^2+z^2)ds=\frac{1} {3}R^2\int_Lds=\frac{1} {3}R^22\pi R$
$\int_Lxds=\int_Lyds=\int_Lzds=\frac{1} {3} \int_L(x+y+z)ds=\frac{1} {3} \int_L 0ds=0$
原式$=\int_Lx^2ds+2\int_Lyds=\frac{2\pi} {3}R^3$

例11

计算$\iint_{\Sigma}zdS$，其中$\Sigma$为曲面$z=\sqrt{x^2+y^2} \,$在柱体$x^2+y^2\leqslant 2x$内部分
解：$\Sigma$在$xoy$平面上的投影区域为$D:x^2+y^2\leqslant 2x,\Sigma:z=\sqrt{ x^2+y^2 },dS=\sqrt{1+\frac{ x^2 } { x^2+y^2 }+\frac{ y^2 } { x^2+y^2 } }d \sigma = \sqrt{2} d \sigma$
原式 $= \iint_{D} \sqrt{ x^2+y^2 } \sqrt{2} d \sigma =
\sqrt{2} \int_{- \frac{\pi} {2} }^{\frac{\pi} {2} }d \theta \int_0^{2 \cos^2 \theta} r^2dr =\\
\frac{16} {3} \sqrt{2} \int_0^{\frac{\pi} {2} } \cos^3 \theta d \theta =
\frac{16} {3} \sqrt{2} \frac{2} {3} = \frac{32} {9} \sqrt{2} \,$.

插入

求质心

$(\overline{x},\overline{y},\overline{z}),$密度函数$\mu=f(x,y,z)$

均质物体质心为形心，上方被积函数为$x(y/z)$下方为1，

求转动惯量

求立体$\Omega$密度函数$\mu=f(x,y,z)$连续，求$\Omega$对直线$L$的转动惯量$I_L$

求引力

设质点A$(x_o,y_o,z_o)$质量m，设立体$\Omega$的密度函数$\mu=f(x,y,z)$连续，求$\Omega$对A的引力$\vec{F} \,$
$\vec{F}=F_x\vec{i}+F_y\vec{j}+F_z\vec{k} \,$
$F_x=km\iiint_{\Omega} \frac{(x-x_o)f(x,y,z)} {r^3}d\Omega$
$F_y=km\iiint_{\Omega} \frac{(y-y_o)f(x,y,z)} {r^3}d\Omega$
$F_z=km\iiint_{\Omega} \frac{(z-z_o)f(x,y,z)} {r^3}d\Omega$

例12

求$z=x^2+y^2$与$z=1$围成均质物体立体$\Omega$的重心
解：由立体$\Omega$关于$yoz$平面、$zox$平面对称且均质知，$\overline{x}=0,\overline{y}=0$

$\overline{z}=\frac{\iiint_{\Omega}zdV} {\iiint_{\Omega}1dV}=$ $\frac{\int_0^1z \pi zdz} {\int_0^1\pi zdz}=$ $\frac{\pi \tfrac{1} {3} } {\pi \tfrac{1} {2} }=$ $\frac{2} {3} \,$

例13

$A(x_o)$质量为$m$,有一个位于区间$[a,b]$上的细棒L，密度函数$\mu=f(x)$,求L对A点的引力
解：$\vec{F}=F_x\vec{i},F_x=km\int_a^b\frac{(x-x_o)f(x)} {|x-x_o|^3}dx=km\int_a^b\frac{f(x)} {|x-x_o|(x-x_o)}dx$

例14

在密度$\mu=1$，半径为R的球体内挖去一半径为$\frac{R} {2} \,$且与球内切的球，球剩余部分对两球公共直径的转动惯量
解:建立空间坐标系，令公共直径位于Oz方向上，
设大球式子为$x^2+y^2+z^2=2Rz(\rho=2R\cos\phi)$，小球为$x^2+y^2+z^2=Rz(\rho=Rcos\phi)$
