# 4. 为什么要建立布里渊区

固体的能带理论中，各种电子态按照它们的波矢分类。在波矢空间中取某一倒易阵点为原点，作所有倒易点阵矢量的垂直平分面，这些面波矢空间划分为一系列的区域：其中最靠近原点的一组面所围的闭合区称为第一布里渊区。

如果在kkk 空间内把原点和所有倒格子格矢GGG 之间的联线的垂直平分面都画出来，kkk 空间被分成许多区域，在每个区域内EEE 对kkk 是连续变化的，而在这些区域的边界处E（k）E（k）E（k）函数发生突变 这些区域常称为布里渊区，可以清楚直观的了解波矢所蕴含的信息。

# 5. 布里渊区边界的物理意义是什么

在这个位置上，光波会发射最强的散射，形成XRDXRDXRD 的增强反射，只要波矢落在布里渊区边界上，就可以满足布拉格反射条件。

# 6. 叙述布里渊区与倒格子的关系

布里渊区是在倒格子空间中以原点为中心的部分区域。从倒格子空间原点，作与最近邻倒格点、次近邻倒格点、再次近邻倒格点的连线，再画出这些连线的垂直平分面 (布拉格面), 面所围成的区域根据一定的规律可以合并为各级布里渊区。

包含原点的多面体所围区域就是第一布里渊区，与第一布里渊区相邻、且与第一布里渊区体积相等的区域为第二布里渊区；与第二布里渊区相邻、且与第一布里渊区体积相等的区域为第三布里渊区...... 同时，第一布里渊区实际是倒格子空间的Wigner−SeitzWigner-SeitzWigner−Seitz 原胞，第nnn 布里渊区是从第n−1n-1n−1 布里渊区出发只通过一个布拉格面就可以到达的点的集合。

# 7. 叙述布里渊区边界与倒格子的关系

在物理学中，倒易点阵是另一个点阵的傅立叶变换。在一般应用中，该第一晶格通常是实空间中的周期性空间函数。正格子存在于实际空间中并且是人们通常理解的物理晶格，倒格子存在于倒易空间。因此，倒格子的倒格子是原来的正格子，两种晶格互为傅里叶变换。

从倒格子空间原点，作与最近邻倒格点、次近邻倒格点、再次近邻倒格点的连线，再画出这些连线的垂直平分面，这些平面构成所有布里渊区的边界，每一级布里渊区可以通过 6 所述的法则一步步求出。
